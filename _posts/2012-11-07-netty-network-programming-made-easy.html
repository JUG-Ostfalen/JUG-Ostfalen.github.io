Es ist eine bewährte Strategie, während der Mittagspause den Fitnessraum für eine halbe Stunde aufzusuchen ( entspricht 6km auf dem Laufband) und danach an einem völlig leeren Buffett einzukehren. Jetzt widme ich mich Dingen, von denen ich bisher nix gehört habe und das Ding heißt Netty!

Speaker ist Norman Maurer (Red Hat)

Netty is another Network Framework!

Features of Netty 4:

  • Komplett asynchron
    • all IO operations don't block
    • Transports: NIO, OIO, AIO, Local, Embedded
    • Protocols: TCP, UDP, SCTP* (Linux)
    • ChannelHandler (Type Safe, Byte / Message based, ....)
    • Codecs: HTTP, WebSockets, SPDY,SSL...
    • ChannelPipeline
  • High-performant
  • Simple unified API
  • Listener & Callback pattern supported
  • Zero-File Copy

Alles sehr Low-Level aber durchaus interessant, sich in die Niederungen der Channel und NIO zu begeben. Vert.X benutzt übrigens auch Netty.
